# 🎓 College Student Performance Analytics Dashboard

> **An End-to-End Data Analytics Project using Python, SQL Server, and Power BI**

---

## 📌 Project Overview

Educational institutions generate large volumes of student-related data, including academic records, attendance, certifications, internships, projects, placement information, and career readiness assessments. However, this data is often scattered across different systems and remains underutilized for academic planning and decision-making.

This project demonstrates an **end-to-end Data Analytics workflow** by transforming raw educational ERP data into meaningful insights using **Python, SQL Server, and Power BI**.

The project enables institutions to:

- Monitor academic performance
- Analyze skill development
- Evaluate internship participation
- Assess placement readiness
- Identify academically at-risk students
- Support data-driven institutional decision-making

---

## 🎯 Business Problem

Educational institutions require a centralized analytical solution to understand how academic performance, attendance, certifications, internships, and placement activities influence student success.

This project answers the following business question:

> **How can educational institutions leverage student data to identify performance trends, improve student engagement, enhance placement readiness, and support data-driven academic decision-making?**

---

## 🎯 Project Objectives

- Analyze academic performance trends
- Evaluate attendance patterns
- Study certification participation
- Analyze internship involvement
- Assess placement readiness
- Identify high-risk students
- Develop interactive dashboards for decision-making.
